# Profile Card

A simple, responsive profile card built with HTML and CSS.

## Features

- Clean layout
- Circular profile image
- Contact form (name, email, message)
- Mobile-friendly design

## Files

- `index.html`: Structure
- `styles.css`: Styling
- `README.md`: Project info
